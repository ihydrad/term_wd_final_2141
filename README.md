# AVR Thermometer

Термометр на AVR с датчиком температуры DS18B20 (1-Wire) и 4-разрядным 7-сегментным индикатором.

## Возможности

- Поддержка до 3 датчиков DS18B20 по шине 1-Wire
- Переключение между датчиками кнопкой (INT0)
- Отображение температуры со знаком (отрицательные значения)
- Индикация ошибки CRC (`rEr`) при сбое данных
- Автоотключение дисплея через ~20 сек, включение по кнопке
- Watchdog Timer для защиты от зависания

## Железо

| Сигнал | Пин |
|--------|-----|
| 7-seg сегменты | PORTB |
| Цифра 1 | PD6 |
| Цифра 2 | PD4 |
| Цифра 3 | PD3 |
| Цифра 4 | PD5 |
| 1-Wire (DS18B20) | PB0 |
| Кнопка | INT0 |

## Сборка

Проект открывается в **CodeVisionAVR** (`main.cproj`).
