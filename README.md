# AVR Thermometer

Термометр на AVR с датчиком температуры DS18B20 (1-Wire) и 4-разрядным 7-сегментным индикатором.

## Возможности

- Поддержка 2 датчиков DS18B20 по шине 1-Wire (автопоиск по Search ROM)
- Переключение между датчиками кнопкой (INT0)
- Отображение температуры со знаком (отрицательные значения)
- Индикация ошибки CRC (`rEr`) при сбое данных
- Автоотключение дисплея через ~20 сек, включение по кнопке
- Watchdog Timer для защиты от зависания

## Железо

| Сигнал | Пин |
|--------|-----|
| 7-seg сегменты | PORTB |
| Цифра 1 | PD6 |
| Цифра 2 | PD4 |
| Цифра 3 | PD3 |
| Цифра 4 | PD5 |
| 1-Wire (DS18B20) | PB0 |
| Кнопка | INT0 (PD2) |

## Сборка

Тулчейн: **avr-gcc** (через PlatformIO или WinAVR).

```bash
make        # → main.elf + main.hex
make flash  # → прошивка через USBasp
make clean  # → удалить артефакты сборки
```

Программатор по умолчанию — `usbasp`. Изменить: `make flash PROGRAMMER=arduino`.
